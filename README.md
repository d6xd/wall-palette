## Overwiew

This repository provides a **fully synchronized desktop theming experience**

The goal is simple:  
**make your desktop look clean, cohesive, and aesthetic with minimal effort.**

---

## Included

Each theme pack may contain:

-  Wallpapers (color-matched)
-  Terminal configs
-  Editor themes (e.g. micro)
-  App-specific configs (where available)

---

## Supported Color Schemes

- Catppuccin
- Gruvbox
- Tokyo Night
- (more coming soon...)

## Get Started

- Clone this repository:
```bash
git clone <link>
```

- Choose a theme!:
```bash
cd wall-palette/
```

- The wallpaper is located in the path `cd Colorschemes/` And then you choose a wallpaper for the desired color scheme, there are also configs
